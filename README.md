### PYTHON

> This is a series of Python projects defined to practice this language.
> * **vector:** Implements a Vector class. It has been good to practice class creation and operator overloading.
> * **cooking:** Implements two classes for handling cooking recipes and another one with a command line interface for storing, accessing, and deleting recipes.
> * **sos:** It's a quite straightforward project; it takes some parameters from the input line and encrypts or decrypts them to Morse code.
> * **guess:** Might be the simplest one. It generates a random number and asks the user for input until it guesses the number correctly.
